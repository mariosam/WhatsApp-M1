********************************************************************************
*** WHATSAPP MODULE - READ ME (english)
********************************************************************************

*** Objectives

The objective of this module is to present a floating contact button on the pages
from the store for the customer to take their doubts via whatsapp.

*** Version 1.0.0

What we have in this version:
- possibility to include a WhatsApp contact number;
- possibility to configure the floating button (color and position);
- possibility to include messages in the button and in the contact box;
- possibility to include URL of the page that generated the contact;
- possibility to enable/disable button according to device;
- possibility to customize visuals using CSS and JS.

*** Tested

Version: 1.9.3.9 Community Edition

PHP: 5.6.40

Theme: RWD

*** Install

Move the <WhatsApp> folder to the directory:
app/code/community/MarioSAM/ (create the directory if it doesn't exist)

Move the <mariosam_whatsapp.xml> file to the directory:
app/design/frontend/base/default/layout/

Move the <whatsapp> folder to the directory:
app/design/frontend/base/default/template/mariosam/ (create the directory if it doesn't exist)

Move the <MarioSAM_WhatsApp.xml> file to the directory:
app/etc/modules/

Move the pt_BR, en_US, etc. translation files/folders to:
app/locale/

Move the <css/mariosam/whatsapp> folder to the directory:
skin/frontend/base/default/

Move the <images/mariosam/whatsapp> folder to the directory:
skin/frontend/base/default/

After moving all files, you have to log out and log in to the backend again.

Access the Magento panel and navigate to:
System > Configuration > MARIO SAM > WhatsApp
